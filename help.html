User Guide to An imporved crypto system

This is help centre for our application.

Main Screen

This is the home page of our application. On this screen you get option to either encrpyt or decrpyt file. Also there is small button in bottom left corner of the screen by which you can choose theme of the application like modern GUI we try to implement both light and dark theme in our system.

Encryption Screen

This is Encrpytion Screen.On this screen you will get option to either encrpyt file by selecting Browse File option. Or you can put text in text box provided and then enter your password in right corner, then click on encrpyt file/text to encrypt and a confirmation message box will appear. Along with a window which shows your text after encryption. NOTE:this window will only open if you have entered text in text box. And aslo your file is saved into the folder named Encrypted_file with name which you will see on confirmation message box.

Decryption Screen

This is Decryption Screen. On this screen you will get option to either Decrpyt file by selecting Browse File option. Or you can put text in text box provided (provided that the decrypted text is from our system only, any other decrypted text from other algorithm will may produce error) and then enter your correct password in right corner else you will get and warning message if your password is incorrect, then click on decrpyt file/text to decrypt and a confirmation message box will appear. Along with a window which shows your text after decryption. NOTE:this window will only open if you have entered text in text box. And aslo your file is saved into the folder named Decrypted_file with name which you will see on confirmation message box.

Help Screen

This is Help Center Screen. Help center or help screen will appear if you clicked on small i icon present on bottom right corner of the screen. This will led you to our website, or can help you to give quick tutorial how to use the application.

Browse file Screen

This is Browse file screen Screen. This is basically open when you click on Browse File option either in Encrpyt Screen or in Decrypt Screen (NOTE: that the above screen shot was took on linux machine and may look different on you operating system deending on your native graphics) In this you can select file which you want to encrpyt. Uptill now only few file types are supported by our sytem i.e .text, .md, and other text based further work to de done.
